# Система финансового учета (fin_count)

Система для учета финансовых операций, подотчетных средств, кассовых операций и отчетности.

**ПРОЕКТ В РАЗРАБОТКЕ**

- Функциональность может быть неполной
- Могут присутствовать ошибки и недоработки
- API и структура данных могут изменяться без предварительного уведомления
- Не рекомендуется использовать в production-окружении

## Технологии

- **Backend:** Django 5.2.8, Django REST Framework
- **Frontend:** React 18, Material-UI, Vite
- **База данных:** SQLite (разработка) / PostgreSQL (production)
- **Python:** 3.14+

## Установка и запуск

### Требования

- Python 3.14+
- Node.js 25+
- npm или yarn

### Backend

```bash
# Создание виртуального окружения
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows

# Установка зависимостей
pip install -r requirements.txt

# Настройка переменных окружения
cp .env.example .env
# Отредактируйте .env файл, установив SECRET_KEY и другие параметры

# Применение миграций
python manage.py migrate

# Создание суперпользователя
python manage.py createsuperuser

# Запуск сервера разработки
python manage.py runserver
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend будет доступен по адресу http://localhost:3000

## Основные возможности

- Управление справочниками (валюты, кассы, сотрудники, статьи доходов/расходов)
- Документы: приход/расход денежных средств, выдача подотчетных средств, авансовые отчеты
- Отчетность
- REST API для интеграции
- Веб-интерфейс на Material Design

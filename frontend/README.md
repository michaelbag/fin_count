# Frontend для системы финансового учета

Frontend приложение на React + Material-UI с возможностью просмотра и редактирования документов.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

## Сборка для production

```bash
npm run build
```

## Особенности

- Material Design UI (Material-UI)
- Просмотр и редактирование документов
- Управление справочниками
- Интеграция с Django REST Framework API
- Валидация форм
- Пагинация списков

## Структура

- `src/pages/` - Страницы приложения
- `src/components/` - Переиспользуемые компоненты
- `src/services/api.js` - API клиент для работы с backend
